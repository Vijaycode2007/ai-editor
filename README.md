# AI Editor ✨

An AI-powered text editor built using **React**, **FastAPI**, and **Google Gemini API** to help users write better, faster, and more effectively.

The editor provides intelligent writing assistance through grammar correction, spell checking, autocomplete suggestions, and advanced text analysis.

---

## 🚀 Features

### ✅ Grammar Correction

* Detects and corrects grammatical errors in text.
* Supports multiple languages.

### ✅ Spell Checking

* Identifies misspelled words.
* Suggests accurate corrections.

### ✅ AI Autocomplete

* Provides smart autocomplete suggestions while typing.
* Helps users write faster and more efficiently.

### ✅ Text Analysis

* Analyzes sentences and paragraphs.
* Generates improved and more insightful text.

### ✅ Multi-language Support

* Works with text written in different languages using Google Gemini AI.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Vijaycode2007/ai-editor.git
cd ai-editor
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
```

Create a `.env` file:

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

Start the backend server:

```bash
uvicorn main:app --reload
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
